# How to Write COMMENTS ?

### Single Line Comment

`--` You need **two dashes followed by a space**:

```sql
-- select * from table_all_users;
```

OR

&#x20;`#`  You just need a hash

```sql
# select * from table_all_users;
```

### Multi Line Comments

```sql
/* 
select * from table_all_users; 
select * from table_all_students;
*/
```
